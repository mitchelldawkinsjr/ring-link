# Wrestling Talent Marketplace MVP Technical Spec

## Core Features
- Wrestler profiles
- Promotion profiles
- Search/discovery
- Booking workflows
- Messaging
- Verified reviews

## Architecture
- Next.js frontend
- Laravel backend
- MySQL
- Redis
- Docker
- Cloudflare R2

## Verified Reviews
- One review per completed booking
- Promotion-only reviews
- Moderation workflow
